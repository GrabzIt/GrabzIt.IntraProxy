The IntraProxy enables GrabzIt to call websites on your internal network, or local machine. One example would be to securely take screenshots of locally hosted websites.

Set Up

  1. Ensure this machine is connected to the Internet. If it is not connect to the Internet and restart the IntraProxy.
  2. Allow port {Port} on this machines firewall
  3. Configure port forwarding.Please do not ask us how to do this as each router is different, information on configuring your router should be available on the Internet.
    1. Allow port {Port} on the routers firewall
    2. Forward all requests to {Port} to this machines IP Address: {IPAddress}

Generate a Call

The IntraProxy redirects all requests to your external IP address and port to the internal website. You can use the below tool to generate a IntraProxy URL.

http://{RemoteIPAddress}:{Port}/http://localhost/GrabzItDemo/handler.php

Further Information

More information can be found in our support system. For security reasons the GrabzIt IntraProxy only allows calls from our servers if you want a more flexible version, please contact us.
